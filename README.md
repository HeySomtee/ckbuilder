# CKB Learning Journey

A weekly study of [Nervos CKB](https://docs.nervos.org/) where each topic is paired
with a small TypeScript replica that mirrors my mental model of the concept.

## Structure

```
src/
  week1/
    cell-model/      # Generalized UTXO: Cells, OutPoints, Transactions, Scripts
reports/
  week-1.md          # Weekly write-up
```

## Weekly Index

| Week | Theme | Report | Code |
|------|-------|--------|------|
| 1 | Fundamentals — Cell Model | [reports/week-1.md](reports/week-1.md) | [src/week1/cell-model](src/week1/cell-model) |

## Run

```bash
npm install
npm run week1
```
